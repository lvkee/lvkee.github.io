Markdown语法

① Markdown标题

在Markdown中，在文字的前面加#表示标题，最低加一个#，最多加六个#，分别表示一级标题到六级标题。

eg.

显示效果：

这是一个一级标题

这是一个二级标题

这是一个三级标题

这是一个四级标题

这是一个五级标题
这是一个六级标题

无序列表：

列表嵌套：

只需要在子列表中的每个选项后添加四个空格即可。

⑧ Markdown表格

设置表格的对齐方式：

  • -: 设置内容和标题栏居右对齐

  • :- 设置内容和标题栏居左对齐。

  • ：-： 设置内容和标题栏居中对齐。

eg.

显示效果：
| name | Github |
| ——— | —— |
| Chris Liu | lvkee |
